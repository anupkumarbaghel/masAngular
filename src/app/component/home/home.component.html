MAS? (Material At Sight)

  • There are several P.H.E departmant in a city
  • and each have one or more store where their material stored
  • Like nut, bolt, handpump
  • Material can be submitted or issued from the store
  • Indent and Measurement Book required for submission and issue of material from store
  • All this operation handled by store incharge
  • Store incharge also need to provide MAS Report on dimand means (Material present at store)

Application Overview

  • This is a store management system specifically built for P.H.E
  • MAS process automated with help of this application
  • Indent and Measurement book filled in the system
  • With the help of Indent and Measurement book system can produce different report such as MAS etc

Application Planned Workflow

  • On the top there is an admin
  • Admin can create one or more store
  • Assign key to each store and provide Key to store incharge
  • Store incharge can operate the store by filling Indent and Measurement book of that store
  • As system have Indent and Measurement it can generate MAS report
